Bienvenido a la Price Service API

Esta API te permite consultar precios de productos en función de la fecha, el producto y la marca.

🛠 ¿Cómo usar la API?

Realiza una petición GET a /api/prices con los siguientes parámetros:

Ejemplo de URL:

/api/prices?applicationDate=2020-06-14-10.00.00&productId=35455&brandId=1

📌 Parámetros

  • applicationDate: Fecha y hora de consulta (formato YYYY-MM-DD-HH.MM.SS).
  • productId: ID del producto.
  • brandId: ID de la marca.

📋 Ejemplos de consulta

Consulta 1:

/api/prices?applicationDate=2020-06-14-16.00.00&productId=35455&brandId=1

Consulta 2:

/api/prices?applicationDate=2020-06-14-21.00.00&productId=35455&brandId=1
